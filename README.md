## Instructions on running locally

**Clone this repo:**

```
  git clone https://github.com/g-czajk/Music-player.git
```
**Go to the project directory:**

```
   cd Music-player
```
**Install dependencies:**

```
   npm install
```
**Start dev server:**

```
   npm run start
```
